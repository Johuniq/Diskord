# Diskord Legends

Generate unique, rarity-based Discord profile cards with holographic effects. This open-source project lets you create and share Discord trading cards based on account age and flags.

## Features

- Holographic Discord profile cards
- Rarity tiers and class variants
- Dynamic Open Graph images
- Modern Next.js + Tailwind CSS UI

## Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/Johuniq/diskord.git
   cd diskord
   ```
2. Install dependencies:
   ```bash
   pnpm install
   ```
3. Run the development server:
   ```bash
   pnpm dev
   ```
